# T01 龙头战法策略

## 策略概述
本策略旨在通过量化分析，每日筛选出最具潜力的涨停股作为次日观察标的，并在次日开盘前给出具体的交易建议。

## 执行流程
1.  **T日 20:00**: 执行 `evening_analysis.py`。
    -   调用API获取当日所有涨停股数据。
    -   结合龙虎榜、资金流、板块热度等多维度指标进行综合评分。
    -   选出前5名优质股票，保存至 `next_day_candidates.json`。
2.  **T+1日 09:25**: 执行 `morning_decision.py`。
    -   读取 `next_day_candidates.json` 中的候选股票。
    -   调用API获取这些股票的竞价数据（开盘涨幅、量比、换手率等）。
    -   根据竞价表现对候选股进行最终打分和排序。
    -   生成最终交易建议（买入标的、仓位、理由）。
3.  **T+1日 09:28**: 执行 `notify_user.py`。
    -   将最终交易建议通过预设方式（如邮件、消息）推送给用户。

## 依赖
- Python 3.8+
- requests
- schedule (用于本地测试)
