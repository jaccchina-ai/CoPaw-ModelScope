## 概率预测模块开发

### 模块文件
- `T01_probability_predictor.py` - 概率预测主模块
- `T01_enrich_sector_info.py` - 板块信息补充工具

### 数据需求
- 最小样本：每板块5笔交易
- 中置信度：每板块20笔交易
- 高置信度：每板块50笔交易
- 预测门槛：至少3个板块达到中置信度

### 当前状态
- 历史交易记录（2026-01-01~01-28）无法从stockAPI获取板块信息
- 保留现有数据，等待新数据积累
- 未来交易会自动保存板块信息

### 运行方式
```python
from T01_probability_predictor import run_probability_prediction
result = run_probability_prediction()

# 自动判断数据是否足够
if result['status'] == 'insufficient_data':
    print(result['message'])
else:
    # 输出预测结果
    for sector, pred in result['predictions'].items():
        print(f"{sector}: {pred['probability']*100:.1f}% ({pred['confidence']})")
```

### 情绪周期判断逻辑
```
得分 = 涨停数量分(0-40) + 连板高度分(0-30) + 封单强度分(0-20) - 炸板扣分(0-10)

阶段划分：
- 恢复初期 (0-25分): 仓位10%
- 恢复中期 (25-50分): 仓位30%
- 上升期 (50-70分): 仓位50%
- 高潮期 (70-85分): 仓位30%（降仓）
- 退潮期 (85-100分): 仓位0%（空仓）
```

### 风控评估逻辑
```
总风险分 = 解禁风险(0-30) + 减持风险(0-25) + 炸板风险(0-20) + 高位风险(0-25) + 情绪风险(0-30)

风险等级：
- 高风险(≥50分): 建议回避
- 中风险(25-49分): 降低仓位
- 低风险(<25分): 正常参与
```
