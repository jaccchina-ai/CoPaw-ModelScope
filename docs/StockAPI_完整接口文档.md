## 1. 基础数据

### 1.1 交易日历
- **接口地址**: `/v1/base/tradeDate`
- **更新时间**: 交易日上午9点
- **请求参数**:
  | 参数名 | 类型 | 必须 | 说明 | 示例 |
  |--------|------|------|------|------|
  | tradeDate | string | 是 | 交易时间 | 2021-11-09 |
- **返回参数**:
  | 参数名 | 类型 | 说明 |
  |--------|------|------|
  | isTradeDate | int | 是否为交易日：0-否，1-是 |

### 1.2 A股列表
- **接口地址**: `/v1/base/all`
- **更新时间**: 交易日收盘后30分钟
- **请求频率**: 2次/天
- **请求参数**: 无
- **返回参数**: 所有A股股票代码和名称
- **建议**: 本地保存，每天请求一次

### 1.3 所有指数代码
- **接口地址**: `/v1/base/ZSCode`
- **更新时间**: 交易日15:30
- **请求参数**: 无
- **返回参数**: 所有指数代码列表

### 1.4 股票/板块日K线行情
- **接口地址**: `/v1/base/day`
- **更新时间**: 交易日15:30
- **请求参数**:
  | 参数名 | 类型 | 必须 | 说明 | 示例 |
  |--------|------|------|------|------|
  | code | string | 是 | 股票代码 | 000001 |
  | startDate | string | 是 | 开始日期 | 2023-01-01 |
  | endDate | string | 是 | 结束日期 | 2023-12-31 |
  | cycle | int | 否 | 周期：100-日线，101-周线，102-月线 | 100 |
- **返回参数**:
  | 参数名 | 类型 | 说明 |
  |--------|------|------|
  | code | string | 代码 |
  | time | string | 时间 |
  | open | string | 开盘价 |
  | high | string | 最高价 |
  | low | string | 最低价 |
  | close | string | 收盘价 |
  | volume | string | 成交量 |
  | amount | string | 成交额 |
  | turnoverRatio | string | 换手率 |
  | changeRatio | string | 涨跌幅 |

### 1.5 ST股列表
- **接口地址**: `/v1/base/stList`
- **更新时间**: 交易日15:30
- **请求参数**: 无

### 1.6 龙虎榜单
- **接口地址**: `/v1/base/topList`
- **更新时间**: 交易日15:30
- **请求参数**:
  | 参数名 | 类型 | 必须 | 说明 | 示例 |
  |--------|------|------|------|------|
  | date | string | 是 | 交易时间 | 2021-11-09 |
- **返回参数**:

- **type参数说明**:
  - 1: 按照抢筹委托金额排序
  - 2: 按照抢筹成交金额排序

  - 4: 按照抢筹涨幅排序

- **返回参数**:
  | 参数名 | 类型 | 说明 |
  |--------|------|------|
  | code | string | 代码 |
  | name | string | 名称 |
  | openAmt | long | 开盘金额 |
  | qczf | double | 抢筹涨幅 |
  | qccje | string | 抢筹成交额 |
  | qcwtje | long | 抢筹委托金额 |

## 使用建议

1. **频率限制**: 大部分接口40次/秒，情绪周期10次/秒
2. **本地缓存**: A股列表、板块代码等基础数据建议本地存储
3. **时间窗口**: 
   - 竞价数据只在9:26可用
   - 实时数据只在9:30-15:00可用
   - 历史数据在15:30后更新
4. **错误处理**: API返回400可能是参数错误或非交易时间
5. **Token安全**: 不要在公开场合泄露Token
